maya-bridge
==

A maya plugin designed to connect Autodesk Maya with any graphics or game engine of your choice. By using a shared memory buffer, it facilitates seamless data synchronization between Maya and your selected engine, making it easier to integrate Maya's powerful level design and modeling tools into your game development workflow.

[![License](https://img.shields.io/badge/license-BSD--2%20clause-blue.svg)](https://bkaradzic.github.io/bgfx/license.html)

Goals:

 - Efficiently sync data between Maya and desired engine using a shared memory buffer.
 - Instantly see changes made in Maya in your chosen engine.
 - Supports integration with a variety of game engines and graphics platforms.
